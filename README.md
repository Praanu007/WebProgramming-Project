# Web Programming Project

## 📖 Overview
This repository contains a front-end Web Programming Project. It is a multi-page website built using standard web technologies (HTML, CSS, and JavaScript) to showcase various web development concepts, including page navigation, dynamic task management, and responsive design.

## ✨ Features
* **Multi-Page Layout:** Includes dedicated pages for Home, About, Department, Gallery, Tasks, and Contact.
* **Task Management:** Dynamic task tracking features powered by vanilla JavaScript (`tasks.js` and `task-pages.js`).
* **Component-Based UI:** Reusable navigation bar (`navigationbar.html`) for consistent layout across multiple pages.
* **Styled UI/UX:** Custom CSS styling (`styles.css`) paired with scalable vector graphics (SVG) for visual elements.

## 🛠️ Technologies Used
* **HTML5:** Semantic structure and page layout.
* **CSS3:** Styling and responsive design.
* **JavaScript (Vanilla):** Dynamic functionality, DOM manipulation, and interactive features.
